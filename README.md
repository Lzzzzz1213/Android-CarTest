# 驾考题库
**前排提示**：本程序使用“阿凡达数据”提供的“驾照题库”接口，开源代码不提供该接口，请需要的各位自行申请！运行请用真机运行，模拟器会有网络错误！  

---  

接口位置在代码中有注明，请仔细阅读！  

---  

已知BUG：手机旋转会导致停运或布局错误！（自行修改不可旋转即可）
